## Especificaciones Técnicas

### Open Graph Image
- Tamaño: 1200x630px (1.91:1)
- Formato: JPG
- Calidad: 85%
- Tamaño máximo: 8MB

### Favicon
- ICO: 16x16, 32x32, 48x48px
- PNG: 16x16, 32x32px
- Apple Touch: 180x180px

### Imágenes de Contenido
- Ancho máximo: 1200px
- Formato: JPG (fotos) / PNG (gráficos)
- Alt text obligatorio
- Lazy loading implementado

## Notas de Implementación
- Todas las imágenes deben tener alt text descriptivo
- Implementar lazy loading para rendimiento
- Optimizar para WebP cuando sea posible
- Comprimir imágenes sin perder calidad visible
